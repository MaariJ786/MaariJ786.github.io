    Iteration - keywords

    here are some of the keywords on iteration

    Iteration is looping a statement to meet a definite argument.

    WHILE and FOR are the terminologies normally used for iteration.

    Definite Iteration: Looping for a fixed amount of times

    Indefinite Iteration: Looping for an unknown amount of times but will stop when it meets the argument.

    Infinite Iteration: A loop that never ends until the user breaks it or when the system runs out of memory.

    Trace Tables - keywords

    Trace tables are a way to dry run your algorthims.

    Syntax Error: This is if there is a "grammatical" mistake in the code.

    Logic Error: This is if the comparison or the mathematical operation is written wrong. This will run but output will be wrong.

    Run Time Error: This is if the program runs for too long and the computer runs out of memory.

    The only error you will find in Pseudocode/Flowchart Algorithm is a Logic Error as it is the only error that depends on the programmer to be certain about.

    Trace Tables: These tables can be used to Dry Run your algorithm. It is a table of variables and is used to check how those variables change throughout the program.
