    • Chapter 2.1 Iteration

    While Loops

    The while construct consists of a block of code and a condition/expression. The condition/expression is evaluated, and if the condition/expression is true, the code within the block is executed. This repeats until the condition/expression becomes false. Because the while loop checks the condition/expression before the block is executed, the control structure is often also known as a pre-test loop. an example of a while loop is shown below.

    int x = 0;

    while (x < 5)

    { printf ("x = %d\n", x); x++; }

    For Loops

    A for loop is a repeated code for a certain amount of times. like the while, For loops run until the condition is met.Unlike many other kinds of loops, such as the while-loop, the for-loop is often distinguished by an explicit loop counter or loop variable. This allows the body of the for-loop (the code that is being repeatedly executed) to know about the sequencing of each iteration. For-loops are also typically used when the number of iterations is known before entering the loop. For-loops are the shorthand way to make loops when the number of iterations is known, as every for-loop could be written as a while-loop.

    Definate iteration

    A definate iteration is where we know the code will end after it meets its parameter

    Indefinate iteration

    We know in a indefinate loop the programme will stop however we dont know when it will end.

    Infinite iteration

    In an infinite loop the programme will never stop it will keep running until the computer or the programme crashes.
